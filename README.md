# PyFyTable
A table software implemented in Python.

---
## Help
First open Main.py:
``` bash
python3 ./luffy/Main.py
```
### Write
```
[ WRITE
>> rc,what you want to write
```
### READ
```
[ READ
>> rc
```
### Save
```
[ SAVE
```
### Exit
```
[ EXIT
```
